先輸入git branch -m oldName newName
將local端的branch改為新的
並且把舊的Branch從remote端刪掉
git push origin -d oldName
之後解除newName與origin oldBranch的連結
git branch --unset-upstream <new_name>
最後將新branch推上去
git push -u origin newBranch

方法二
再改完local端的branch name之後
git push origin :oldName newName
即可改掉remove branch name
